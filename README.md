## Tools versions

node version: 12.18.4 - 14.15.4
npm version: 6.14.6 - 6.14.11

## Pre requisites

1. Windows 10 (64-bit).
2. Resolution of monitor 1080p (24 inches recommended).
3. Google chrome Version 86.0 (64-bit).
4. Browser zoom recommended 100%.
